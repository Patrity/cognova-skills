# Cognova Skills

Community skills for [Cognova](https://github.com/Patrity/cognova) - a personal knowledge management system with Claude Code integration.

## Installing Skills

Skills from this library can be installed directly from the Cognova web UI:

1. Navigate to **Skills** > **Library**
2. Browse available skills
3. Click **Install**

Or manually copy a skill directory to `~/.claude/skills/`.

## Available Skills

| Skill | Description | Secrets Required |
|-------|-------------|-----------------|
| [system-info](./system-info/) | Gather system information (OS, CPU, memory, disk, network) | None |
